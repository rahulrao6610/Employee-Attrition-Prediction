# 🏢 Employee Attrition Prediction using Machine Learning
## 📌 Overview
Employee Attrition Prediction is a Machine Learning project designed to help Human Resource (HR) departments identify employees who are at risk of leaving the organization. The system analyzes employee-related factors such as age, department, job role, monthly income, work-life balance, overtime, business travel, job satisfaction, and years at the company to predict attrition.
The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, model comparison, hyperparameter tuning, feature importance analysis, and a Streamlit-based web application that provides real-time predictions along with HR recommendations.

## 🚀 Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Logistic Regression
- Random Forest Classifier
- Gradient Boosting Classifier
- Hyperparameter Tuning (GridSearchCV)
- Feature Importance Analysis
- Model Serialization using Joblib
- Interactive Streamlit Dashboard
- Real-Time Employee Attrition Prediction
- HR Decision Support Recommendations

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Streamlit
- Joblib
  
## 📊 Project Outcome
The application predicts whether an employee is likely to leave the organization and provides HR professionals with actionable insights to improve employee retention and support data-driven decision-making.
